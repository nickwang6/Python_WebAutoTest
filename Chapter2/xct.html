第3节：元素定位-xpath、css

目标

  • [x] 熟悉Xpath定位策略
  • [x] 熟悉css定位策略

为什么要学习Xpath、CSS定位

  • 在实际项目中标签没有id、name、class属性

  • id、name、class属性值为动态获取，随着刷新或加载失败

1.什么是Xpath

  1. XPath即为XML Path 的简称，它是一种用来确定XML文档中某部分位置的语言。

  2. HTML可以看做是XML的一种实现，所以Selenium用户可以使用这种强大的语言在Web应用中定位元素。

    XML：一种标记语言，用于数据的存储和传递。 后缀.xml结尾

    提示：Xpath为强大的语言，那是因为它有非常灵活定位策略

表达式 描述
nodename 选取此节点的所有子节点
/ 从根节点选取
// 从匹配选择的当前节点选择文档中的节点，而不考虑其位置
. 选取当前节点
.. 选取当前节点的父节点
@ 选取属性

2.1 路径（绝对路径、相对路径）

​ 绝对路径：从最外层元素到指定元素之间所有经过元素层级路径，以/开始

​ 相对路径：从第一个符合条件元素开始（一般配合属性来区分），以//开始

from  selenium import  webdriver
from  time import  sleep

driver=webdriver.Chrome()
#url
driver.get('https://www.baidu.com')
sleep(2)

#最大化
driver.maximize_window()

#通过相对路径进行定位
driver.find_element_by_xpath("//input[@id='kw']").send_keys('自动导航')
#通过绝对路径进行定位百度一下
#driver.find_element_by_xpath('/html/body/div[1]/div[1]/div[5]/div[2]/div/form/span[2]/input').click() 容易失效

3.CSS定位

​

3.1什么是CSS？

  1. CSS（Cascading Style Sheets）是一种语言，它用来描述HTML和XML的元素显示样式； (css语言书写两个格式：
                        1. 写在HTML语言中<style type="text/css">... )
                                            2. 写在单独文件中 后缀.css
  2. 而在CSS语言中有CSS选择器(不同的策略选择元素)，在Selenium中也可以使用这种选择器； 提示：
    1. 在selenium中极力推荐CSS定位，因为它比XPath定位速度要快
    2. css选择器语法非常强大，在这里我们只学习在测试中常用的几个

​

3.2CSS定位常用策略

​

  1. id选择器

    说明：根据元素id属性来选择 格式：#id 如：#userA <选择id属性值为userA的所有元素>

   from  selenium import  webdriver
   from  time import  sleep

   #实例化浏览器
   driver=webdriver.Chrome()
   #url
   driver.get("https://www.baidu.com/")
   #最大化
   driver.maximize_window()

   driver.find_element_by_css_selector('#kw').send_keys('demo')
   sleep(2)
   driver.close()

​ 2.class选择器

​ 说明：根据元素class属性来选择 ​ 格式：.class 如：.telA <选择class属性值为telA的所有元素>

​ 

from  selenium import  webdriver
from  time import  sleep

#实例化浏览器
driver=webdriver.Chrome()
#url
driver.get("https://www.baidu.com/")
#最大化
driver.maximize_window()

driver.find_element_by_css_selector('.s_ipt').send_keys('demo')
sleep(2)
driver.close()

3.元素选择器

​ 说明：根据元素的标签名选择 ​ 格式：element 如：input <选择所有input元素>

from  selenium import  webdriver
from  time import  sleep

#实例化浏览器
driver=webdriver.Chrome()
#url
driver.get("https://www.baidu.com/")
#最大化
driver.maximize_window()

driver.find_element_by_css_selector("input[name='wd']").send_keys('demo')
sleep(2)
driver.close()

4.属性选择器

​ 说明：根据元素的属性名和值来选择 ​ 格式：[attribute=value] 如：[type="password"] <选择所有type属性值为password的值>

from  selenium import  webdriver
from  time import  sleep

#实例化浏览器
driver=webdriver.Chrome()
#url
driver.get("https://www.baidu.com/")
#最大化
driver.maximize_window()

driver.find_element_by_css_selector("[name='wd']").send_keys('demo')
sleep(2)
driver.close()

5.层级选择器

​ 说明：根据元素的父子关系来选择 ​ 格式：element>element 如：p>input <返回所有p元素下所有的input元素> ​ 提示：> 可以用空格代替 如：p input 或者 p [type='password']

from  selenium import  webdriver
from  time import  sleep

#实例化浏览器
driver=webdriver.Chrome()
#url
driver.get("https://www.baidu.com/")
#最大化
driver.maximize_window()

driver.find_element_by_css_selector("form > span > input").send_keys('demo')
sleep(2)
driver.close()

3.4CSS总结

选择器 例子 描述
#id #kw id选择器，选择id="kw"的所有元素
.class .s_ipt class选择器，选择class="s_ipt"的所有元素
element input 选择所有input元素
[attribute=value] [name='wd'] 选择type="wd"的所有元素
element>element form > span > input 选择form > span > input元素

4.Xpath与CSS类似功能对比

5.定位（另一种写法）

​ 第二种方法使用By类的封装的方法,所以需要导入By类包


"""

    1. driver.find_element(By.CSS_SELECTOR,'xxx').send_keys("xxx")
    2. driver.find_element(By.XPATH,'//*[@id="xxx"]').send_keys('xxx')
    3. driver.find_element(By.ID,"xxx").send_keys("xxx")
    4. driver.find_element(By.NAME,"xxx").send_keys("xxx")
    5. driver.find_element(By.CLASS_NAME,"xxx").send_keys("xxx")
    6. driver.find_element(By.TAG_NAME,'xxx').send_keys("xxx")
    7. driver.find_element(By.LINK_TEXT,'xxx').click()
    8. driver.find_element(By.PARTIAL_LINK_TEXT,'xxx').click()

"""


from selenium import  webdriver
from  selenium.webdriver.common.by import By
from  time import  sleep

#实例化浏览器
driver=webdriver.Chrome()
#url
driver.get('https://www.baidu.com/')
#通过另外一种方式进行元素定位

driver.find_element(By.ID,"kw").send_keys('webdriver')
sleep(2)
driver.close()
