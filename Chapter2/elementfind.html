第2节：元素定位

第2节：元素定位

目标

  • [x] 了解元素各种定位方法

  • [x] 掌握id、name、class_name、tag_name、link_text、partial_link_text 定位的使用

1.为什么要学习元素定位方式？

  • 让程序操作指定元素，就必须先找到此元素
  • 程序不像人类用眼睛直接定位到元素
  • WebDriver提供了八种定位元素方式

2.WebDriver元素定位方式

​

  • id、name、class_name：为元素属性定位
  • tag_name：为元素标签名称
  • link_text、partial_link_text：为超链接定位(a标签)
  • Xpath：为元素路径定位
  • Css：为CSS选择器定位

WebDriver demo地址

2.1 id定位

​ 说明：HTML规定id属性在整个HTML文档中必须是唯一的，id定位就是通过元素的id属性来定位元素； ​ 前提：元素有id属性。

#这里选用百度地址
from  selenium import  webdriver
from  time import  sleep

#这里使用谷歌浏览器   93.0.4577.63   驱动版本为 93.0.4577.63
driver=webdriver.Chrome()
#url
driver.get('https://www.baidu.com/')
sleep(5)
#全屏
driver.maximize_window()
#通过id元素进行访问
driver.find_element_by_id('kw').send_keys("自动化测试")
#关闭浏览器
driver.close()

2.2 name定位

​ 说明：HTML规定name属性来指定元素名称，因此它的作用更像人名，name的属性值在当前文档中可以不是唯一的 ，name定位就是 根据元素name属性来定位

#这里选用百度地址
from  selenium import  webdriver
from  time import  sleep

#这里使用谷歌浏览器   93.0.4577.63   驱动版本为 93.0.4577.63  
driver=webdriver.Chrome()
#url
driver.get('https://www.baidu.com/')
sleep(5)
#全屏
driver.maximize_window()
#通过name元素进行访问
driver.find_element_by_name('wd').send_keys("自动化测试")
#关闭浏览器
driver.close()

2.3 class_name定位

​ 说明：HTML规定了class来指定元素的类名，用法和name、id类似

#这里选用百度地址
from  selenium import  webdriver
from  time import  sleep

#这里使用谷歌浏览器   93.0.4577.63   驱动版本为 93.0.4577.63  
driver=webdriver.Chrome()
#url
driver.get('https://www.baidu.com/')
sleep(5)
#全屏
driver.maximize_window()
#通过class_name元素进行访问
driver.find_element_by_class_name('s_ipt').send_keys("自动化测试")
sleep(1)
#关闭浏览器
driver.close()

2.4 tag_name定位

​ 说明：HTML本质就是由不同的tag(标签)组成，而每个tag都是指同一类，所以tag定位效率低， ​ 一般不建议使用；tag_name定位就是通过标签名来定位；

#这里选用百度地址
from  selenium import  webdriver
from  time import  sleep

#这里使用谷歌浏览器   93.0.4577.63   驱动版本为 93.0.4577.63
driver=webdriver.Chrome()
#url
driver.get('https://www.taobao.com/')
sleep(5)
#全屏
driver.maximize_window()
#这里通过定位tag_name进行元素定位
try:
    driver.find_element_by_tag_name('input').send_keys("月饼")
except Exception as e:
    print(e)
#关闭浏览器
driver.close()

2.5 link_text定位

​ 说明：link_text定位与前面4个定位有所不同，它专门用来定位超链接文本（标签）.

#这里选用百度地址
from  selenium import  webdriver
from  time import  sleep

#这里使用谷歌浏览器   93.0.4577.63   驱动版本为 93.0.4577.63
driver=webdriver.Chrome()
#url
driver.get('https://www.baidu.com/')
sleep(5)
#全屏
driver.maximize_window()
#这里通过link_text 进行定位

driver.find_element_by_link_text('新闻').click()
#关闭浏览器
driver.close()

​ 说明：partial_link_text定位是对link_text定位的补充，partial_like_text为模糊匹配；link_text全部匹配
