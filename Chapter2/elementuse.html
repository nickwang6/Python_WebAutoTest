第4节：元素操作|浏览器操作方法

目标

  • [x] 掌握WebDriver常用的元素操作方法

  • [x] 掌握WebDriver常用的浏览器操作方法

1.为什么要学习操作元素的方法？

​ 1.需要让脚本模拟用户给浏览器指定元素输入值

​ 2.需要让脚本模拟人为删除元素的内容

​ 3.需要让脚本模拟点击按钮

2.元素常用操作方法

​ 1.clear() 清除文本

​ 2.send_keys() 模拟输入

​ 3.click() 单击元素

from selenium import  webdriver
from  time import sleep

#实例化浏览器
driver=webdriver.Chrome()
#url
driver.get('https://www.baidu.com/')
#元素定位
t=driver.find_element_by_id('kw')
t.send_keys('123456')
sleep(2)
t.clear()
t.send_keys('csdn')
sleep(2)
driver.close()

3.浏览器常用方法


"""


1. maximize_window()                最大化 --> 模拟浏览器最大化按钮
2. set_window_size(100,100)            浏览器大小 --> 设置浏览器宽、高(像素点)
3. set_window_position(300,200)     浏览器位置 --> 设置浏览器位置
4. back()                             后退 --> 模拟浏览器后退按钮
5. forward()                         前进 --> 模拟浏览器前进按钮
6. refresh()                         刷新 --> 模拟浏览器F5刷新
7. close()                            关闭 --> 模拟浏览器关闭按钮(关闭单个窗口)
8. quit()                            关闭 --> 关闭所有WebDriver启动的窗口


"""


from selenium import  webdriver
from  time import sleep

#实例化浏览器
driver=webdriver.Chrome()

#url
driver.get("https://www.baidu.com/")

#浏览器最大化
driver.maximize_window()

#元素定位搜素
driver.find_element_by_id("kw").send_keys('testDemo')
driver.find_element_by_id('su').click()
sleep(2)

#后退
driver.back()
sleep(2)
#前进
driver.forward()

#关闭
driver.close()

4.WebDriver其他常用方法

"""

1. size                 返回元素大小
2. text                 获取元素的文本
3. title                 获取页面title
4. current_url            获取当前页面URL
5. get_attribute("xxx") 获取属性值;xxx：要获取的属性
6. is_display()            判断元素是否可见
7. is_enabled()            判断元素是否可用


"""

from selenium import  webdriver
from  time import  sleep


#无头模式设置
option=webdriver.ChromeOptions()
option.add_argument('headless')
driver=webdriver.Chrome(options=option)


#url
driver.get('https://www.baidu.com/')
#最大化
driver.maximize_window()
#Y元素定位 返回输入框的大小
s=driver.find_element_by_id('kw').size
print(s)
#定位新闻 返回text
s1=driver.find_element_by_link_text("新闻").text
print(s1)
#返回页面title
print(driver.title)
#获取当前url
print(driver.current_url)
