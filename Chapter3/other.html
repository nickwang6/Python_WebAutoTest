1.处理表单

​ 说明：表单处理通过元素定位即可；多选框单选框通过元素定位+click()

from  selenium import  webdriver
from  time import  sleep

#浏览器实例化
driver=webdriver.Chrome()
#url
driver.get('http://sahitest.com/demo/formTest.htm')
#隐式等待
driver.implicitly_wait(5)
#最大化
sleep(2)
#处理弹框
a=driver.switch_to.alert
#接受
a.accept()
#页面最大化
driver.maximize_window()

#元素定位form
driver.find_element_by_css_selector('body > form > table > tbody > tr:nth-child(2) > td:nth-child(2) > input').send_keys('name')

#处理checkbox
driver.find_element_by_css_selector('body > form > input[type=checkbox]:nth-child(16)').click()
sleep(2)
#处理radiobutton
driver.find_element_by_css_selector('body > form > input[type=radio]:nth-child(26)').click()

#结束
driver.close()

2.处理下拉框

​ 说明：导入selenium工具包中select包

​

方法/属性 方法/属性描述
select_by_value() 根据值选择
select_by_index() 根据索引值选择
select_by_visible_text() 根据文本选择
deselect_by_value 根据值反选
deselect_by_index 根据索引反选
deselect_by_visible_text 根据文本反选
deselect_all 反选所有
options 所有选项
all_selected_options 所有选中选项
first_selected_option 第一个选择选项 
from selenium import webdriver
from  time import  sleep
from selenium.webdriver.support.select import Select

#实例化浏览器
driver=webdriver.Chrome()
#隐式等待
driver.implicitly_wait(5)
#url
driver.get('http://sahitest.com/demo/selectTest.htm')
#最大化
driver.maximize_window()

#元素定位  索引定位
i=driver.find_element_by_id('s1Id')
Select(i).select_by_index(1)
#通过value定位
v=driver.find_element_by_id('s2Id')
Select(v).select_by_value('o1')
#通过文本定位
t=driver.find_element_by_id('s3Id')
Select(t).select_by_visible_text('o1')
#结束
driver.close()

3.处理弹框

​ 页面上的弹框有三种

​ alert：用来提示

​ confirm：用来确认

​ prompt：输入内容

方法/属性 方法/属性描述
accept（） 接受
dismiss（） 取消
text 显示文本
send_keys 输入内容 
from  selenium import  webdriver
from time import  sleep

#实例化浏览器
driver=webdriver.Chrome()
#隐式等待
driver.implicitly_wait(5)
#url
driver.get('http://sahitest.com/demo/alertTest.htm')
sleep(2)
#最大化
driver.maximize_window()
#元素定位
driver.find_element_by_name('b1').click()
a=driver.switch_to.alert
a.accept()
