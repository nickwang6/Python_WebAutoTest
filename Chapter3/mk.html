  • [x] 了解Webdriver操作鼠标方法

  • [x] 了解Webdriver操作键盘方法

1.鼠标事件

​ 说明：现在Web产品中提供了丰富的鼠标交互方式，如：双击、悬停、拖拽等功能，作为Web产品测试框架，需要应对这些Web产品应用场景，所以在WebDriver类库中封装了相应的鼠标交互方法。

1.1 鼠标执行 perform

​ 说明：在ActionChains类中所有提供的鼠标事件方法，在调用的时候所有的行为都存储在ActionChains类中, 而perform()方法就是执行所有ActionChains中的行为

1.2鼠标右击-context_click()

​ 说明：在ActionChains类中的鼠标右键方法是个已知的bug，因为它能点击右键出现菜单，但无法选择右键菜单选项，因此方法了解即可

from  selenium import  webdriver
from  time import  sleep
#导入ActionChains包
from selenium.webdriver.common.action_chains import ActionChains


#实例化浏览器
driver=webdriver.Chrome()
#url
driver.get('http://sahitest.com/demo/clicks.htm')
sleep(2)
#最大化
driver.maximize_window()

#css元素定位
element=driver.find_element_by_css_selector('body > form > input[type=button]:nth-child(13)')
ActionChains=ActionChains(driver).context_click(element).perform()

1.3 鼠标单击-double_click()

​ 说明：模拟双击鼠标左键操作

from  selenium import  webdriver
from  time import  sleep
#导入ActionChains包
from selenium.webdriver.common.action_chains import ActionChains


#实例化浏览器
driver=webdriver.Chrome()
#url
driver.get('http://sahitest.com/demo/clicks.htm')
sleep(2)
#最大化
driver.maximize_window()

#css元素定位
element=driver.find_element_by_css_selector('body > form > input[type=button]:nth-child(10)')
ActionChains=ActionChains(driver).click(element).perform()

1.4 鼠标拖动-drag_and_drop()

​ 说明：模拟鼠标拖动动作，选定拖动源元素释放到目标元素

from  selenium.webdriver.common.action_chains import ActionChains
from selenium import  webdriver
from  time import sleep

#实例化浏览器
driver=webdriver.Chrome()
#url
driver.get('http://sahitest.com/demo/dragDropMooTools.htm')
#source定位原始模块位置 target为拖动位置
source=driver.find_element_by_css_selector('#dragger')
target=driver.find_element_by_css_selector('body > div:nth-child(4)')
Action=ActionChains(driver)
Action.drag_and_drop(source,target).perform()

1.5鼠标悬停move_to_element()

​ 说明: 模拟鼠标悬停在选中的元素

from  selenium.webdriver.common.action_chains import ActionChains
from selenium import  webdriver
from time import sleep

#实例化浏览器
driver=webdriver.Chrome()
#url
driver.get('http://sahitest.com/demo/mouseover.htm')
sleep(2)

#元素定位
e=driver.find_element_by_css_selector('body > a:nth-child(2) > span')
Action=ActionChains(driver)
Action.move_to_element(e).perform()

2.键盘操作

​ 说明： ​ 1). 模拟键盘上一些按键或者组合键的输入 如：Ctrl+C 、Ctrl+V； ​ 2). WebDriver中对键盘的操作都封装在Keys类中

2.1常用的键盘操作

"""

1. send_keys(Keys.BACK_SPACE)删除键（BackSpace）
2. send_keys(Keys.SPACE)空格键(Space)
3. send_keys(Keys.TAB)制表键(Tab)
4. send_keys(Keys.ESCAPE)回退键（Esc）
5. send_keys(Keys.ENTER)回车键（Enter）
6. send_keys(Keys.CONTROL,'a') 全选（Ctrl+A）
7. send_keys(Keys.CONTROL,'c')复制（Ctrl+C）

"""
from selenium import  webdriver
from  selenium.webdriver.common.keys import Keys
from time import sleep

#实例化浏览器
driver=webdriver.Chrome()
#url
driver.get('https://www.baidu.com')
#元素定位输入框
driver.find_element_by_id('kw').send_keys('python')
driver.find_element_by_id('su').send_keys(Keys.ENTER)
sleep(2)
driver.close()
